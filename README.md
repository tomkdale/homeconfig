# homedir

Tom's dotfiles held here, do this on new machine install

```
$ mkdir github/
$ git clone https://github.com/tomkdale/homedir ./github/
$ cp github/homedir/.[Xa-z]* .
$ bin/,configurevim
$ rm -r github/homedir
```

 - git@github.com:agkozak/zsh-z.git
